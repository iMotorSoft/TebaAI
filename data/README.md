# TebaAI Data Policy

This directory is prepared for corpus inputs, processed outputs and generated
reports.

## Rules

- Do not commit real books, corpora or heavy files.
- `data/source/` is for raw input material.
- `data/processed/` is for derived or normalized data.
- `data/reports/` is for generated evidence and reports.
- Only README files, `.gitkeep` files and explicitly approved tiny examples
  should be tracked.
- Secrets, private datasets and licensed material must not be committed.

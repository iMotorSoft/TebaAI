# TebaAi Docs

Documentation root for TebaAi non-runtime material.

## Structure

- `adr/`: architecture decision records.
- `arquitectura/`: architecture narrative that is not a living invariant.
- `estrategia/`: product and platform strategy.
- `negocio/`: business context.
- `ux/`: product experience and interface decisions.
- `analisis-tecnico/`: technical analysis not tied to active runtime code.
- `templates/`: reusable documentation templates.
